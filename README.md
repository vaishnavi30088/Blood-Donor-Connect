🩸 Blood-Donor-Connect

Blood-Donor-Connect is a web-based application designed to help people quickly find blood donors during emergencies. The system connects donors and recipients by allowing users to register, search for available donors, and contact them easily.

This platform aims to reduce the time required to locate compatible blood donors and improve accessibility to life-saving blood donations.

🚀 Features

User registration and login

Donor registration with blood group and location

Search donors by blood group

View donor contact details

Simple and user-friendly interface

Secure backend system for storing donor information
